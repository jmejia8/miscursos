Visualización de Datos

Descripción del Curso

Este curso tiene como objetivo aprender el lenguaje de programación Julia para visualización de datos. La idea principal es estudiar las bases de Julia desde trabajar en la consola, hasta la creación de interfaces gráficas para la visualización de datos estadísticos.

Material del Curso

  • Lineamientos del curso

  • Conceptos básicos de Julia (notebook)

  • Arreglos y Diccionarios (notebook)

  • Usando DataFrames (notebook)

  • Introducción a Visualización e Interactividad (notebook)

  • Ejemplo de Interfaz (notebook)

Conjuntos de datos:

  • Iris (CSV)

  • Pima Indian Diabetes (CSV, descripción)

  • COVID-19 reportados en México (descripción):

    • Hasta 30 de agosto de 2022 (CSV).

    • Datos actualizados (CSV).

  • Neuro con datos faltantes (CSV).
